# Markdown2HTML

## This is a tool to convert `.md` files into `.html` files.

Here is the current features:

-   Headings conversion
-   Unordered / Ordered lists (with `-` and `*`)
-   Simple texts
-   Bold and emphasis texts (with `**` and `__`)
-   MD5 comversion (with `[[ ]]`)
-   Case insensitive `c` removal (with `(( ))`)
